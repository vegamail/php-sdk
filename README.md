# Vegamail PHP-SDK

PHP library for the Vegamail API.

BETA ONLY FOR NOW   
Contact us for early access: dev@hadoken.io

## Install

Via Composer

``` bash
$ composer require vegamail/php-sdk
```

## Usage

``` php
$apiToken = 'XXXXX'
$client = new Vegamail\Client($apiToken);
```

## Tests

```php
php composer tests
```
